**Microsoft Azure** ist die Cloud‑Computing‑Plattform von Microsoft.

Im Paper wird die Supply‑Chain‑Planung für Serverbereitstellungen in Azure als reales Anwendungsbeispiel für das [[optiguide-framework]] verwendet.

Die zugrunde liegende Supply Chain umfasst Prozesse wie:

- Nachfrageprognosen
- Hardwarebeschaffung
- Planung von Serverbereitstellungen
- Versand von Hardware an Rechenzentren

Die Optimierung wird im sogenannten **Intelligent Fulfillment System (IFS)** durchgeführt. Dieses System entscheidet unter anderem:

- welcher Hardwarelieferant verwendet wird
- wann Server versendet werden
- in welchem Datacenter sie installiert werden

OptiGuide ermöglicht es Planern, Fragen zu diesen Entscheidungen direkt in natürlicher Sprache zu stellen.
