Machine Learning ist laut Quelle keine Hype-Technologie mehr. Drei Viertel der deutschen Unternehmen nutzen ML aktiv oder führen es ein. Besonders größere und mittlere Firmen verwenden mehrere ML-Anwendungen parallel.

Dies deutet auf eine zunehmende Standardisierung von ML im Unternehmenskontext hin.
